## my-tomato-timer-serve

一个简单的任务管理器，欢迎访问 [https://x.withyan.cn/tomato/](https://x.withyan.cn/tomato/) 查看效果。

### 项目说明

* Springboot: v2.1.6 RELEASE
* Redis: 5.0.8
* MySQL: 5.7.29
* Swagger-ui: 2.x

### 打包方法（war）

```bash
mvn package -Dmaven.test.skip=true
```

### 小结

主要提供两方面服务：注册登录与任务管理。简单的一个小项目。
